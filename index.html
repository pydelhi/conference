PyDelhi Conference

What is PyDelhi Conference?

PyDelhi Conference, the premier conference in North India on using and developing the Python programming language is conducted annually by the PyDelhi community. It attracts the best Python programmers from across the country and abroad.

The main aim of the event is to bring together the whole Python community.  In one place including established/startup companies, experts, beginners, global entrepreneurs under one roof.

We will be inviting experts from various fields to showcase how python is used by them, talk about new and upcoming tech.
